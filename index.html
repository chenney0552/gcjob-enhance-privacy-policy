Privacy Policy for GC Jobs Enhancer

Last Updated: March 2024

1. Introduction

GC Jobs Enhancer ("we", "our", or "the extension") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard your information when you use our Chrome extension.

2. Information Collection

2.1 Job Data

  • We collect and store job listings data from the GC Jobs website locally in your browser
  • This includes job titles, descriptions, requirements, and URLs
  • All data is stored using Chrome's local storage API

2.2 User Settings

  • Azure OpenAI configuration (optional)
  • User preferences and settings
  • Favorite job listings

2.3 Technical Data

  • Basic usage statistics
  • Extension performance metrics
  • Error logs for troubleshooting

3. Data Storage and Security

3.1 Local Storage

  • All data is stored locally on your device
  • No data is transmitted to external servers except when using AI features
  • Data is automatically cleared when you clear your browser data

3.2 AI Feature Data Handling

  • When using Azure AI features, job data is temporarily transmitted to Azure OpenAI
  • No personal information is sent to AI services
  • AI analysis results are stored locally only

4. Data Usage

We use the collected data to:

  • Provide job filtering and search functionality
  • Enable keyboard navigation features
  • Store your favorite job listings
  • Improve extension performance
  • Fix bugs and technical issues

5. Third-Party Services

5.1 Azure OpenAI

  • Used only when AI job matching feature is activated
  • Processes job data according to Microsoft's privacy standards
  • No personal data is shared with Microsoft/Azure

6. User Rights

You have the right to:

  • Clear all stored data at any time
  • Disable any feature of the extension
  • Choose not to use AI features
  • Remove the extension completely

7. Data Retention

  • Job data is retained until you clear your browser data
  • Settings are retained until you uninstall the extension
  • Cached data is automatically cleared when changing pages

8. Children's Privacy

This extension is not intended for use by children under 13 years of age.

9. Changes to Privacy Policy

We may update this Privacy Policy from time to time. We will notify users of any material changes through the extension's update notes.

10. Contact Information

For any questions or concerns about this Privacy Policy, please contact us through the Chrome Web Store support channel.

11. Consent

By installing and using this extension, you consent to the collection and use of information as described in this Privacy Policy.

This extension is not affiliated with, endorsed by, or sponsored by the Government of Canada or any of its agencies.
